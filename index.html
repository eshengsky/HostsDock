HostsDock

Store, manage and switch your hosts quickly.

Features

Local&remote schema

Support to create local hosts or connect to the remote hosts file.

Syntax highlighting

Automatically identify IP, domain and comments in hosts and highlighting them.

Powerful editor

Provide copy, paste, cut, search, replace and undo/redo editing functions.

Quick comments

Quick comment one line by clicking the # button or press Ctrl + / .

Fold grammar

Original fold grammar #region to create folding block in hosts.

IP test

Support for testing IPs in hosts with ping command.

System tray

Support the system tray and tray menu to quickly switch hosts.

Export hosts

Support for exporting the selected hosts schemes to share to others.

Multi platform

Support for key operating system: Windows, MacOS and Linux.
